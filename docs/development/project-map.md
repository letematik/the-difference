# Project Map — The Difference

## Resumen general
Este proyecto es un sitio web estático con Astro que funciona como una plataforma artística, conceptual y sensorial.

## Páginas principales
- index: portada y dashboard principal.
- log: bitácora o registro del proyecto.
- le-tematik: espacio cultural y temático.
- nada-brahma: proyecto de atmósfera y presencia.
- texvn: sección de tecnología y visión.
- quantumlab: laboratorio conceptual y experimental.
- pensamiento-autista: reflexión y filosofía.
- saiyain-do: sección de práctica o propuesta.
- ars-tekne: enlace entre arte y técnica.
- quiron-theatre: exploración performativa o teatral.

## Carpetas de recursos por página
Cada página puede tener subcarpetas internas para:
- images
- music
- sfx
- video
- videos
