# Contexto del proyecto The Difference

## Proyecto
The Difference es un sitio web estático con Astro. Tiene una identidad visual oscura, minimalista y conceptual, con páginas temáticas y una navegación basada en experiencias sensoriales.

## Objetivos del proyecto
- Mantener una estética coherente y experimental.
- Usar imágenes, audio y carruseles como parte del lenguaje visual.
- Organizar cada sección con recursos propios: imágenes, audio, video y efectos.
- Mantener el sitio fácil de extender y respaldar.

## Estructura recomendada
- public/{pagina}/images para imágenes numeradas
- public/{pagina}/music para música
- public/{pagina}/sfx para efectos de sonido
- public/{pagina}/video o videos para material audiovisual

## Estilo de interacción
- El sitio prioriza la atmósfera y la experiencia más que la sobrecarga visual.
- Los carruseles deben ser lentos, continuos y contemplativos.
- La navegación debe sentirse como un recorrido interno.
