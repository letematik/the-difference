# Workflow recomendado para el proyecto

## Flujo simple
1. Trabaja en el sitio con Astro.
2. Guarda cambios en Git.
3. Usa la IA local para revisar ideas, textos y estructura.
4. Mantén los recursos en carpetas separadas por tipo.
5. Haz backups periódicos del repositorio.

## Flujo más profesional
- Usa GitHub como respaldo visible y compartido.
- Usa Docker + Ollama + Open WebUI para trabajar con IA local.
- Usa Qdrant para guardar contexto del proyecto.
- Usa Continue para tener asistencia contextual dentro del editor.
