# Configuración recomendada para Open WebUI

## Objetivo
Usar Open WebUI como interfaz visual para conversar con Ollama y mantener el flujo de trabajo más claro.

## Pasos recomendados
1. Abre http://localhost:3000.
2. Conecta el backend de Ollama.
3. Selecciona un modelo como `qwen2.5:7b`.
4. Usa la IA para revisar prompts, contenidos, ideas visuales y documentación.

## Uso ideal para este proyecto
- Revisar textos del sitio.
- Proponer nuevas secciones o cambios de navegación.
- Ayudar con la organización de recursos y estructura de carpetas.
- Explorar ideas creativas sin salir del entorno local.
